## Check l'app en mode local

```
$ git clone https://github.com/Gyenh/from_scratch_or_not
```
```
$ bundle install --without production
```
```
$ rails server
```
Puis dans la barre d'url du Navigateur web
```
$ localhost:3000
```

ruby '2.5.1'
